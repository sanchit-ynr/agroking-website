# Deployment

## Build
- Install: `npm install`
- Build static export: `npm run build`
- Output directory: `out`

## Cloudflare Pages
- Framework preset: None (static site)
- Build command: `npm run build`
- Build output directory: `out`

## Vercel
- Import repo
- Build command: `npm run build`
- Output directory: `out`

## Local Preview
- `npm run dev` for development at http://localhost:3000
- `npm run build` then serve `out` with any static server for production preview.
